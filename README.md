🚀 Fundamentos de Programação Orientada a Objetos (POO)
Este repositório contém a implementação prática de conceitos fundamentais de Programação Orientada a Objetos, utilizando entidades do mundo real para demonstrar como organizar dados e comportamentos de forma estruturada e profissional.

📋 Objetivos do Projeto
O foco deste projeto é aplicar os pilares da POO para criar sistemas escaláveis, manuteníveis e seguros.

🛠️ Conceitos Aplicados
🏗️ Modelagem de Entidades
Classes e Objetos: Criação de moldes para representar entidades do mundo real, transformando atributos em dados e comportamentos em métodos.

Encapsulamento: Proteção da integridade dos dados através de modificadores de acesso (private) e implementação de métodos Getters e Setters.

⚙️ Lógica e Comportamento
Métodos Especializados: Implementação de lógicas que variam de cálculos simples a validações condicionais complexas, garantindo que cada classe saiba "resolver seus próprios problemas".

🧬 Reutilização e Estrutura
Herança: Criação de hierarquias para compartilhar atributos e métodos comuns entre classes, evitando a duplicidade de código.

Polimorfismo: Capacidade de tratar objetos de diferentes subclasses de forma uniforme, permitindo comportamentos especializados em métodos sobrescritos.

Interfaces (Contratos): Definição de regras que garantem consistência e flexibilidade, permitindo que diferentes classes implementem os mesmos comportamentos de formas distintas.

💻 Tecnologias Utilizadas
Linguagem: Java

Paradigma: Orientado a Objetos (OOP)
